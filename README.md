# 法人提携ポータルサイト

このプロジェクトは、法人提携業務を効率化するためのポータルサイトです。

## GitHub へのデプロイ方法

このプロジェクトは GitHub Actions を使用して GitHub Pages に自動デプロイされるように設定されています。

### 手順

1.  **GitHub リポジトリの作成**:
    GitHub で新しいリポジトリを作成します。
2.  **コードのプッシュ**:
    ローカルのコードを GitHub リポジトリにプッシュします。
    ```bash
    git init
    git add .
    git commit -m "Initial commit"
    git branch -M main
    git remote add origin <あなたのリポジトリURL>
    git push -u origin main
    ```
3.  **GitHub Pages の設定**:
    GitHub リポジトリの `Settings` > `Pages` に移動します。
    -   `Build and deployment` > `Source` を `GitHub Actions` に設定します。
4.  **デプロイの確認**:
    `Actions` タブでデプロイの進捗を確認できます。完了すると、`https://<ユーザー名>.github.io/<リポジトリ名>/` でサイトが公開されます。

## ローカル環境（自分のPC）での実行方法

コマンド操作に慣れていない方でも、VS Code を使えば簡単に実行できます。

### 事前準備
パソコンに **Node.js** がインストールされている必要があります。
[Node.js 公式サイト](https://nodejs.org/) から「LTS（推奨版）」をダウンロードしてインストールしてください。

### 実行手順（VS Code を使用する場合）

1.  **フォルダを開く**:
    ダウンロードしたプロジェクトを解凍し、VS Code でそのフォルダを開きます。
2.  **ライブラリのインストール（初回のみ）**:
    -   VS Code のメニューから **[表示] > [エクスプローラー]** を開きます。
    -   左下の **[NPM スクリプト]** パネルを開きます。
    -   `install` の横にある **再生ボタン（▷）** をクリックします。
3.  **アプリの起動**:
    -   同じく [NPM スクリプト] パネルにある **`dev`** の横の **再生ボタン（▷）** をクリックします。
    -   画面下のターミナルに `http://localhost:3000` と表示されたら、そこを `Ctrl` キーを押しながらクリックするとブラウザでサイトが開きます。

### 修正の反映
VS Code でファイルを編集して保存（`Ctrl + S`）すると、ブラウザの表示も自動的に更新されます。
修正が終わったら、前述の「GitHub へのアップロード方法」に従って更新を反映させてください。
